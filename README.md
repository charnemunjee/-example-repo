# Project Financial calculators

## Description
This python script calculates the monthly repayment on a mortgage bond as well as the final investment amount at the end of an investment period.

The code asks the user whether they want to calculate their monthly repayment on a mortgage bond or a final investment amount at the end of a specified investment period.
If the user wants to calculate their monthly bond repayment, the code asks for the interest rate and the repayment period
If the user wants to calculate their final investment amount, the code asks the user for the interest rate, the investment period and whether simple or compound interest should be used
